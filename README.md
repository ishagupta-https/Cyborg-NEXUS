# 🤖 Cyborg Nexus - Futuristic AI Landing Page

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/ishagupta-https/cyborg-nexus.git
```

2. Navigate to the project folder

```bash
cd cyborg-nexus
```

3. Open the project

```bash
index.html
```

or run with a local server:

```bash
npx serve
```
